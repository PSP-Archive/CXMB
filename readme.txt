a Custom theme plugin for the PSP devices.

Installation:
Copy cxmb to your seplugins folder.
and add this line to your VSH.txt file ( make it the first line )
ms0:/seplugins/cxmb/cxmb.prx 1
or
ef0:/seplugins/cxmb/cxmb.prx 1
Restart the VSH.
